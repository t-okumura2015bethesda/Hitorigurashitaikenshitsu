# 京都市一人暮らし体験室 オンライン内覧サイト

京都市の地域生活支援拠点等事業「京都市一人暮らし体験室」の物件内を、ブラウザ上で手軽に内覧できるWebサイトです。
間取り図上のマーカーをタップすることで、各部屋の実際の様子を写真で確認することができます。

## 🌟 特徴
- **インタラクティブな間取り図**: 各フロアの間取り画像に配置されたマーカーをクリックすると、その場所の写真がスライダー形式で表示されます。
- **Googleスプレッドシート連携**: 写真のURLや説明文、表示座標をスプレッドシートで一括管理しています。
- **モバイルフレンドリー**: スマートフォンでの操作（ピンチイン・アウト、スワイプ）に最適化されたビューアーを搭載しています。
- **スムーズなUI**: データの読み込み待ちを知らせるローディング画面や、ブラウザの「戻る」ボタンに対応したスムーズな操作感を実現しています。

## 🛠 使用技術
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Backend**: Google Apps Script (GAS)
- **Database**: Google Sheets (Google スプレッドシート)

## 🏗 システム構成
本プロジェクトは、以下の仕組みで動作しています。
1. **Google Apps Script**: スプレッドシートのデータを読み取り、JSON形式のAPIとして公開。
2. **JavaScript (fetch API)**: GASが書き出したAPIを叩き、動的にコンテンツを生成。
3. **GitHub Pages**: HTML/CSS/JavaScriptをホスティングし、サイトを公開。



## 📝 備考
このプロジェクトは、障がい児・者相談支援センター「いまじん」の事業「一人暮らし体験室」の利便性向上のために作成されたものです。
